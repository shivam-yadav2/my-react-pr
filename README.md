# my-react-pr

A CLI tool to quickly create a React project from a template repository.

## Installation & Usage

You can use this package in two ways:

### Method 1: Using npx (Recommended - No installation needed)
```bash
npx my-react-pr myapp
```

### Method 2: Install globally first
```bash
npm install -g my-react-pr
my-react-pr myapp
```

## What it does

This tool clones the React project template repository and sets it up in the specified folder name.

## Next Steps

After running the command:
```bash
cd myapp
npm install
npm start
```
